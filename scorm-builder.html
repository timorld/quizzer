Re-upload Mode Experimental

Keep original package ID on export
When ON, the exported SCORM will use the same manifest identifier as the imported package. This does not guarantee ALM preserves learner data — ALM resets tracking records when content is replaced regardless of the ID. May be useful for other platform-level purposes.

AI Assistant Configured on the server — not part of SCORM exports

The AI conversion endpoint is configured server-side via env vars (NVIDIA_API_KEY, NVIDIA_API_ENDPOINT, NVIDIA_API_MODEL). No per-user API key is required. Use the button below to verify the service is reachable.

Remote-Loaded Player Experimental

When ON, exported SCORMs load the player JS and CSS from https://timorld.github.io/quizzer-player/ instead of bundling them inside the ZIP. This makes it possible to push player updates to already-deployed quizzes without re-uploading. Filename gets a _remote suffix so these packages are easy to spot. Default OFF — leave OFF for normal production exports.

Use remote-loaded player on export
While ON, every SCORM download (single, batch, history re-export) is built with remote script tags. The learner's browser must be able to reach GitHub Pages at quiz launch time.

Enable Show Answer Review?

    

      
When Show Answer Review is enabled:

      
    
        
  • A Review Quiz button will appear on the result screen after every attempt — both pass and fail
    • 
        
  • Learners can review the answers they submitted before deciding to retake
    • 
      

      
When this is off, the Review Quiz button is only shown to learners who pass, and their correct answers are revealed in the review.

    

    

      
      
    

  







  

    
Enable Best Score Tracking?

    

      
When best score tracking is enabled, the following applies to this quiz:

      
    
        
  • Learners who have already passed can retake the quiz, subject to your attempts and cooldown settings
    • 
        
  • The LMS always records the highest passing score, not the most recent attempt
    • 
        
  • If a learner retakes and scores lower, their original higher score is preserved
    • 
        
  • The result screen will inform learners that their highest score will be saved
    • 
        
  • The Review Quiz button will show answers in normal mode — correct answers are not revealed, so retakes reflect genuine learning
